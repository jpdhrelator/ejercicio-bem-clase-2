# ejercicio-bem-clase-2

### Antes de Comenzar

Genere una rama con la siguiente estructura nombre-apellido y guarde ahi su avance y luego subalo al finalizar la clase

### Instrucciones Actividad:

1. Análisis : Dibuja en papel o visualiza un cuadro que contenga: una imagen de producto, un título, un precio y un botón de compra.

2. HTML: Escribe la estructura HTML.

- El contenedor padre debe ser el Bloque (ej: .producto).

- La imagen, título y precio son Elementos (ej: .producto\_\_imagen).

- Necesitamos dos tarjetas: una normal y una que esté "Agotada". Para la agotada, usa un Modificador en el bloque padre (ej: .producto--agotado).

3. CSS: Dale estilos.

- El bloque .producto debe tener borde y padding.

- Los elementos deben tener tamaños de fuente y margenes.

- El modificador .producto--agotado debe cambiar el fondo a gris y bajar la opacidad de la imagen.

No olvide crear las carpetas necesarias para esta solicitud.

Los colores y aspecto visual son a libre elección.
